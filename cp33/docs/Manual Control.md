# Manual Controller Quick Guide #

- [The model](https://drive.google.com/drive/folders/1DMvKe5ccxUgHKO4TKUjxAzRSMg9YMGGQ?usp=sharing)

place the whole folder (`simple_model_v2`) it in `objdetection/drone_controller`

To start

```{bash}
./startup.py -m
```

To control:

```
w: forward
s: backward
a: left
d: right
r: upward
f: downward
j: turn left
k: turn right
t: take off
l: land
```
